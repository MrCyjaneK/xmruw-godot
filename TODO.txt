to get started you need /usr/lib/monero_libwallet2_api_c.so, you can grab it:
```
sudo wget https://github.com/MrCyjaneK/monero_c/releases/download/v0.18.3.3-RC48/monero_x86_64-linux-gnu_libwallet2_api_c.so.xz -O /usr/lib/monero_libwallet2_api_c.so.xz
sudo unxz -f /usr/lib/monero_libwallet2_api_c.so.xz
```
or if you want to build it from source:
```
git clone https://github.com/MrCyjaneK/monero_c
cd monero_c
./build_single.sh monero x86_64-linux-gnu
# and move the .so file from release/monero to /usr/lib/monero_libwallet2_api_c.so
# it may fail on archlinux though... so use ubuntu/debian docker if possible
```
